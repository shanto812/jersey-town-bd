# 🛒 E-commerce Website

A modern, responsive, and fully functional e-commerce web application built with clean UI and smooth user experience. This project includes essential features for both users and admin with scalable architecture.

## 🚀 Live Demo

👉 [https://e-commerce-alpha-amber-20.vercel.app/](https://jersey-town.vercel.app/)



## ✨ Features

### ❤️ Wishlist System

* Users can add or remove products to wishlist
* Wishlist data is stored permanently (localStorage / database)
* Works even after page refresh

### 📦 Featured Categories

* All category boxes have equal size (uniform UI)
* Clean and integrated single section design
* Fully responsive layout

### 🛍️ Shop Page (New)

* Dedicated page for all products
* Search functionality 🔍
* Category filter 🗂️
* Wishlist support ❤️
* Add to cart system 🛒
* Mobile & desktop responsive 📱💻

### 🔐 Admin System

* Admin logout redirects to login page
* Login page includes “Back to Home” option
* Secure and structured navigation flow

### 🖼️ Image Upload System

* Direct image upload from device
* No need for image URL
* Instant image preview before upload
